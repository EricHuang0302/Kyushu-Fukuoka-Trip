# 7/7（二）佐賀機場 → 佐賀站午餐 → 福岡

## 主軸
第一天先從佐賀機場搭巴士到 JR 佐賀站，在佐賀站附近吃午餐，再搭 JR 特急到福岡。下午不要排太遠，晚上看博多祇園山笠飾山笠，晚餐吃博多華味鳥水炊き。

## 交通重點

- 佐賀機場到市區的巴士搭乘處在**機場正面玄關正前方**。
- 佐賀機場巴士只有一條路線，方向是前往佐賀市區／佐賀站巴士中心，因此不容易搭錯。
- 7/7 使用方向：**佐賀機場 → 佐賀站巴士中心**。
- 抵達佐賀站巴士中心後，JR 佐賀站就在旁邊，可步行前往。
- 午餐後從 JR 佐賀站搭 JR 特急到博多站。

# 7/8（三）福岡 → 太宰府 → 福岡

## 主軸
第二天把太宰府天滿宮放進來。這天不去門司港，門司港與咖哩本舖燒咖哩整段拿掉。晚上回福岡市區吃肉いち燒肉。

## 交通重點

- 第一推薦：從博多巴士總站搭太宰府ライナーバス「旅人」到太宰府。
- 「旅人」不需要預約；博多巴士總站到太宰府標準車程約 40 分鐘。
- 備案：地鐵到天神後，轉西鐵電車至西鐵二日市，再轉太宰府線到西鐵太宰府站。
